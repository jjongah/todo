# 할일 관리 (To-Do Manager) - 프로젝트 완료

React 기반의 직관적이고 모던한 할일 관리 웹 애플리케이션입니다.

## 🎯 완성된 주요 기능

### 📝 할일 관리
- ✅ 할일 추가/수정/삭제
- ✅ 완료 체크 토글
- ✅ 제목, 메모, 날짜, 시간, 폴더 지정
- ✅ localStorage 자동 저장 (새로고침해도 유지)

### 📅 3가지 뷰 모드

#### 1️⃣ 오늘 뷰
- 블럭 형태로 오늘 할일 목록 표시
- 체크, 수정, 삭제 가능

#### 2️⃣ 이번주 뷰 (NEW!)
- **월~일 7개 박스**로 요일별 구분
- 각 요일에 해당 날짜의 할일 표시
- 오늘 날짜 하이라이트
- 마우스 오버 시 삭제 버튼 표시

#### 3️⃣ 나중에 뷰 - 월별 달력 (NEW!)
- **전체 월 달력** 형태
- ◀ ▶ 버튼으로 **다른 달 이동 가능**
- "오늘" 버튼으로 현재 달로 즉시 이동
- 각 날짜에 최대 3개 할일 표시
- 3개 이상은 "+N개 더" 표시
- 마우스 오버 시 삭제 버튼 표시

### 📁 폴더 시스템
- ✅ 기본 폴더 3개 (학교, 개인, 알바)
- ✅ 커스텀 폴더 추가/수정/삭제
- ✅ 폴더별 파스텔 톤 색상 자동 지정
- ✅ 각 폴더의 미완료 할일 개수 표시

### 📊 대시보드
- ✅ 오늘 날짜와 요일
- ✅ 통계 (전체/완료/남은 할일)
- ✅ 완료율 프로그레스 바
- ✅ 오늘의 할일 미리보기

### 🎨 디자인
- ✅ 심플하고 모던한 UI
- ✅ 흰색 배경 + 파스텔 톤
- ✅ 부드러운 애니메이션
- ✅ 직관적인 UX

### 📱 반응형
- ✅ 데스크톱: 사이드바 + 메인
- ✅ 태블릿: 최적화 레이아웃
- ✅ 모바일: 햄버거 메뉴 + 플로팅 버튼

## 💡 주요 특징

1. **3가지 뷰 모드**: 오늘/이번주(요일별)/나중에(달력)
2. **직관적인 상호작용**:
   - 체크박스: 완료 토글
   - 제목 클릭: 수정 모달
   - × 버튼(hover): 삭제
3. **폴더별 색상 구분**: 파스텔 톤으로 시각적 구분
4. **완전한 반응형**: 모든 디바이스 지원
5. **자동 저장**: localStorage로 데이터 영구 보존
6. **달력 네비게이션**: 이전/다음 달 이동, 오늘로 돌아가기

## 📝 사용 가이드

### 할일 추가
- 상단 "할일 추가" 버튼 또는 플로팅 버튼(+) 클릭
- 제목, 메모, 날짜, 시간, 폴더 입력 후 저장

### 할일 관리
- **완료**: 체크박스 클릭
- **수정**: 할일 제목 클릭
- **삭제**: 할일에 마우스 오버 후 × 버튼 클릭

### 뷰 전환
- **오늘**: 오늘 해야 할 일 목록
- **이번주**: 월~일 요일별 박스로 구분
- **나중에**: 월별 달력으로 전체 보기

### 폴더 관리
- 사이드바에서 "새 폴더" 버튼으로 추가
- 폴더명 위에 마우스 오버 후 수정/삭제
